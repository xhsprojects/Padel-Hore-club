# **App Name**: Padel Hore Leaderboard

## Core Features:

- Match Input Form: Admin input form for match data, including player names, scores, and host designation, with auto-complete for player names.
- Point Calculation Engine: Calculates points based on participation, match results, margin of victory, attendance, and bonus conditions (win streaks, underdog bonus) as detailed in the rules.
- Anti-Abuse Validation: Checks if a player has played more than 2 matches in a day and reduces points for subsequent matches to prevent point farming. Requires tool use.
- Tier Update: Automatically updates player tiers based on total points accumulated.
- Real-time Leaderboard: Displays a sortable leaderboard showing player rankings, win rates and other stats that updates in real-time.
- Player Profiles: Displays individual player statistics including total matches, win rate, tier, and match history.
- Season Toggles: Allows users to view rankings for the current month or all-time rankings.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey sophistication and seriousness of play.
- Background color: Very light gray (#F0F0F0), almost white, to create a clean, uncluttered space.
- Accent color: Lavender (#E6E6FA) to add a touch of elegance and highlight interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a modern, sporty look. Body font: 'Inter' sans-serif for readability.
- Use modern, minimalist icons to represent match statistics and player achievements.
- Clean and structured layout for the leaderboard and player profiles to ensure easy navigation and readability.
- Subtle transitions and animations to indicate data updates and tier changes.
